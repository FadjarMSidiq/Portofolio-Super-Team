Tugas Praktek Kolaborasi dengan Git/Github

Manajemen Proyek

Topik: pembuatan website statis (dengan HTML), dengan tema: Portofolio Super Team
Tiap anggota kelompok (termasuk ketua) menuangkan profile di halaman-halaman website
yang berbeda. Isi profile a.l: nama, foto diri (atau diganti dengan foto/gambar yang
lucu/keren), hobi, keahlian (bahasa, framework, dll), aktivitas terakhir (ada sedikit keterangan
dan foto, foto tidak harus riil, bisa yang keren/lucu saja).
Pembuatan website dilakukan dengan memanfaatkan Github dan Git, minimal dengan 2
siklus.
Siklus-1:
1. Project Manager (PM) membuat repositori sentral di Github dengan file README
dan index.html. Pada index.html dapat diisi dengan menu berupa link-link ke
halaman-halaman lain.
2. Tiap anggota melakukan clone dari repositori sentral ke branch di repo lokal. Branch
diberi nama sesuai dengan nama masing-masing anggota.
3. Tiap anggota membuat file HTML (dengan nama yang sesuai dengan link di
index.html), dengan isi: nama, foto diri (atau diganti dengan foto/gambar yang lucu),
hobi, keahlian (bahasa, framework, dll). Lalu branch di-push ke Github.
4. PM melakukan review dari tiap branch, lalui melakukan merge ke main (master)
branch.
Siklus-2:
1. Tiap anggota melakukan pull dari repositori sentral ke branch di repo lokal untuk
mendapatkan file-file HTML terbaru.
2. Tiap anggota edit file HTML masing-masing dengan menambah konten aktivitas
terakhir (termasuk foto), lalu melakukan push ke repo sentral.
3. PM melakukan review dari tiap branch, lalui melakukan merge ke main (master)
branch.

4. Setiap anggota melakukan pull (clone) dari file-file HTML ke komputer masing-
masing, lalu menjalankan website yang lengkap (di komputer lokal).

Yang dikumpulkan oleh PM (ketua kelompok):
Sebuah file teks berisi link Github website itu.
